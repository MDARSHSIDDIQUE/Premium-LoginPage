# Premium Glass Login Page

A single-page premium login UI built with HTML, CSS, and a tiny JavaScript password toggle. It is designed as a clone-and-customize template.

## Customize

- Change visible text in `src/index.html`, especially `Your Brand`.
- Change colors in the `:root` CSS variables.
- Replace `src/assets/premium-login-bg-v3.png` with your own background image.
- Connect the `<form>` action or JavaScript submit handler to your backend.

## Run

Open `src/index.html` directly in your browser.
